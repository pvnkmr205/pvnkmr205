### 🚀 Featured Projects

#### 🤖 [LLM Triage Agent](https://github.com/pvnkmr205/LLM-Triage-Agent)
> A multi-domain triage agent built at the HackerRank Orchestrate Hackathon using a Router-First architecture and BM25 deterministic retrieval to classify and route support tickets securely while eliminating AI hallucinations.  
> **Tech Stack:** Python, LLMs, BM25 (TF-IDF), rich (CLI), JSON

#### 💬 [IntelliRag - Conversational Assistant](https://github.com/pvnkmr205/Project-Intellirag-Retrieval-Augmented-Generation-)
> An AI-powered Telegram bot leveraging Retrieval-Augmented Generation (RAG) to provide precise, context-driven answers from custom data sources.  
> **Tech Stack:** Python, FastAPI, Groq LLM, FAISS, Docker, Telegram Bot API

#### 🛡️ [PhishNet - AI-Based Threat Detection](https://github.com/pvnkmr205/Phishnet-Ai-Based-Phishing-Website-Detection)
> A cybersecurity Chrome Extension utilizing LightGBM and PySpark to process large-scale URL datasets and proactively secure users against fraudulent websites with 95% accuracy.  
> **Tech Stack:** Python, PySpark, Scikit-learn, LightGBM, JavaScript

#### 💪 [Fit AI – Computer Vision Tracker](https://github.com/pvnkmr205/Project--Fit-Ai)
> An intelligent, camera-based fitness companion that actively monitors human movement, dynamically tracks joint angles, and automates repetition counting with zero latency.  
> **Tech Stack:** Python, OpenCV, MediaPipe, TensorFlow
